## Booking application backend
Backend for nextjs booking application

## Features
1. Register user
2. Login
3. Get current logged-in user (me)
4. Add Hotel
5. Update Hotel
6. Delete Hotel
7. Get Hotel
8. Add Room
9. Update Room
10. Delete Room
11. Get Room

## Technologies used:
- node
- express
- jwt
- mongoose
- zod
## Installation

Make sure you have node and yarn installed.

### Install packages

```bash
yarn
```

### Start local server

```bash
yarn dev
```

This starts the local server at [http://localhost:1337](http://localhost:5173)
